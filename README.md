# Sneakers

https://ev-climb.github.io/sneakers/

## Описание проекта

Интернет-магазин обуви.\
Реализовано:
- подгрузка товаров с mocapi
- добавление в корзину покупок и избранное
- отправка заказа на сервер
- поиск по товарам

- ReactJS + хуки
- React Router
- Axios
- SCSS

- React Skeleton
